import numpy as np

def scaled_dot_product_attention(Q, K, V):
    """Minimal self-attention implementation."""
    d_k = Q.shape[-1]
    scores  = Q @ K.T / np.sqrt(d_k)     # scale
    weights = softmax(scores, axis=-1)  # attend
    return weights @ V                    # context

def softmax(x, axis=-1):
    e_x = np.exp(x - x.max(axis, keepdims=True))
    return e_x / e_x.sum(axis, keepdims=True)

What are the key differences between MHA and GQA in modern LLMs?
✦
Comparison · Attention Variants
  • MHA — Multi-Head Attention: each head has its own Q, K, V projections. High expressivity, high memory cost.
  • GQA — Grouped Query Attention: multiple query heads share K/V heads. Reduces KV-cache size dramatically — used in Llama 3, Mistral.
  • MQA — Multi-Query Attention: extreme case of GQA with a single K/V head. Fastest inference, slight quality trade-off.
